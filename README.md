### Stack: 

#### Frontend
- HTML5
- CSS3
- Bootstrap
- JavaScript
- JQuery

#### Backend
- PHP 7.2
- Laravel 6
- Composer
- PHPUnit

#### Database
- MySQL

---

### Installation

#### Prerequisite
- config file `.env`
- create local database  

#### Backend
- $ composer install  
- $ php artisan key:generate
- $ php artisan elephpants:read
- $ php artisan storage:link

#### Database
- $ php artisan migrate
- $ php artisan db:seed
